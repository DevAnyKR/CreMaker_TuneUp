## 파일 설명

아크릴 휨 방지 강화 블록입니다.

## 출력 조건

- ~180mm_B1.stl 모델은 2개씩 1세트 출력. (측면 강화시 2세트 추가 출력)
  - 단, 손잡이 사용시 블록 1개를 ~180mm_B2.stl 모델로 대체합니다.
- ~100mm_B1.stl 모델은 2개씩 1세트 출력
- 서포트 필요 없는 방향으로 회전
- 구멍은 축소하면 뻑뻑해짐. 0.05~0.1 정도 확장 추천

## 준비물

- M3\*20 볼트 및 너트 18개

## 출력 조건

넓은 면이 바닥으로 오도록 회전
