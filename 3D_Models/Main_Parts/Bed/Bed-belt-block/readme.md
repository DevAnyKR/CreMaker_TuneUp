# Bed-belt-block

+ Y 벨트를 고정하는 블록입니다.

## 효과

### Before

+ 진동 발생 및 블록이 헐거워지면 벨트가 빠짐

### After

+ 볼트로 압착하므로 벨트가 빠지지 않음

## 출력 조건

+ 너트 고정 부분에는 큰 힘이 걸리므로 eSUN PLA+ 이상의 강도를 가진 필라멘트를 권장.
+ 내외벽 두께는 1.2mm 이상, 내부채움은 50% 이상.
