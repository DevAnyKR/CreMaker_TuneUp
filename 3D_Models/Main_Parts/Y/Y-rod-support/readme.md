# Y-rod-support

- Y축에 LM8LUU 베어링을 사용했을 때 간섭되는 부분을 삭제한 블록입니다.

## 효과

### before

- 길어진 LM 베어링이 블록과 충돌하면서 Y축의 이동 거리가 짧아짐

### after

- 충돌 부분을 제거하여 Y축 이동 거리를 원래대로 사용 가능

## 파일 설명

+ B1: 기본 모델(순정 길이의 봉)
+ B2: 관통 모델(순정 보다 긴 봉)

## 출력 조건

+ eSUN PLA+ 추천. 또는 PETG, ABS 등 강도가 높은 재질
+ 프로파일 접촉면이 바닥으로, 세워진 방향으로 배치
