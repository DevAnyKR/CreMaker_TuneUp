# Y-bearing-block

+ Y 벨트 장력 조절 블록입니다.

![](https://github.com/simulz/CreMaker_TuneUp/blob/main/3D_Models/Main_Parts/Y/Y-bearing-block/image/20220512_3D1.png)

## 파일 설명

+ 3.2L: 기본 모델(추천)
+ 3.3L: 홈따기 모델(벨트 좌우 위치 조절용)

## 준비물

+ M6\*20~30mm 볼트
  + 볼트가 길 경우, washer 모델을 출력해서 사용하거나, 금형 스프링(내경 6mm)을 사용할 수 있습니다.
